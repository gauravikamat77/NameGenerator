Instructions:


All allowed routes:


'/register'
Post request for registering new user
Example body: { "name": "Name", "user": "Username", "pass": "Pa$w0rd8" }

'/login'
Post request for loging in
Example body: { "user": "Username", "pass": "Pa$w0rd8" }
Returns an access token

'/refresh'
Get request that returns a new refresh token

'/logout'
Get request that clears the refresh token from cookie



JWT authenticated routes:


'/names'


Allows Create Read Update Delete options

Create:
Post request to create entry
Example body: { "name": "Name", "category": "Category" }
Required role: Admin or Leader or Member

Read all:
Get request that returns all entries
Required role: Admin or Leader or Member or User
Read one:
Get request that returns a single entry from passing ID as parameter
Here: '/names/{parameter}'
Required role: Admin or Leader or Member or User

Update:
Put request that uses ID as an identifier and updates the required field
Example body: { "id": "{ID}", "name": "Name", "category": "Category" }
Required role: Admin or Leader or Member

Delete:
Delete request that uses ID as an identifier and deletes data entry from database
Example body: { "id": "{ID}" }
Required role: Admin or Leader


'/teams'


Allows Create Read Update Delete options

Create:
Post request to create entry
Example body: { "team_id": {ID}, "team_leader": "LeaderName", "team_members": ["MemberName", "MemberName", ...] }
Required role: Admin or Leader

Read all:
Get request that returns all entries
Required role: Admin or Leader or Member
Read one:
Get request that returns a single entry from passing ID as parameter
Here: '/teams/{parameter}'
Required role: Admin or Leader or Member

Update:
Put request that uses ID as an identifier and updates the required field
Example body: { "team_id": {ID}, "team_leader": "LeaderName", "team_members": ["MemberName", "MemberName", ...] }
Required role: Leader or Member

Delete:
Delete request that uses ID as an identifier and deletes data entry from database
Example body: { "team_id": {ID} }
Required role: Admin or Leader
